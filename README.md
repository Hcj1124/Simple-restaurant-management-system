# 餐飲管理系統實作

本專案為資料結構學期專題，旨在實作一套簡易餐廳管理系統。實現了餐廳訂位、菜單管理及訂單處理的快速操作，並透過適當的資料結構設計，優化資料檢索效率。

### Project Overview
模擬餐廳運作流程，重點在於解決高頻率的資料查詢與更新需求。透過設計結構化的資料模型，確保系統在處理大量訂單與菜單資訊時，仍能維持穩定的執行效能。

### Tech Stack
* **語言**: Python
* **開發工具**: VS Code, Git
* **關鍵設計**: 資料結構優化、演算法應用

### Data Structure & Complexity Analysis
核心模組採用 AVL Tree 與 Linked List，以平衡開發複雜度與執行效率：

| 模組功能 | 使用資料結構 | 核心操作 | 時間複雜度 |
| :--- | :--- | :--- | :--- |
| **菜單管理** | **AVL Tree** | 搜尋 / 插入 / 刪除 | $O(\log n)$ |
| **訂單處理** | **Linked List** | 新增 / 移除 (Head) | $O(1)$ 或 $O(n)$ |
| **訂單歷史** | **List (Array)** | 儲存 / 迭代 | $O(1)$ (Append) |
| **管理者帳號** | **Linked List** | 註冊 / 登入搜尋 | $O(n)$ |
